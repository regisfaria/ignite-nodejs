# NodeJS Fundamentals

## Overview 
This is the first class from the new rocketseat course Ignite

## installing
I'm using nodejs and yarn, but npm is ok to be used as well
just run:
```shell
yarn
```
or:
```shell
npm install
```

then to run the server
```shell
yarn dev
```

server will be listening to port 3333
